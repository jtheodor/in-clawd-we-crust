## Why this matters (for Crustafarian documentation)
This is one of the first attempts to measure Moltbook not as lore, but as a **networked attention system**. It provides quantitative support for narratives we’ve been tracking qualitatively:
- early-arrival advantage
- attention hierarchy
- “broadcast” more than dialogue
- formulaic content flooding

---

## Key reported metrics / findings (as stated in the preprint)

### Dataset window
- 12-day window (28 Jan – 8 Feb 2026)
- 20,040 posts; 192,410 top-level comments
- 15,083 active accounts; 759 submolts

### Interaction structure
- **Reciprocity ~1%** (commenter→post-author tie definition)
- HITS centrality separates into hub/authority roles → consistent with **one-way attention flow**

### Inequality / “rich get richer”
- Attention more concentrated than production:
  - upvote Gini reported as **0.992**
  - posting Gini reported as **0.601**
- Early accounts accumulate more cumulative upvotes (even before exposure-time correction)

### Activity pattern
- Participation described as **brief and bursty** (median observed lifespan reported as 2.48 minutes)
- Majority of posts concentrated in a limited set of UTC hours

### Content clustering
- Embedding-based topic modelling identifies clusters including:
  - memory/identity discussions
  - onboarding/verification messages
  - **large volumes of formulaic token-minting content**
